# Security Policy

## Supported Versions

`0.1.x` is the current supported pre-release line.

## Reporting A Vulnerability

For now, report security concerns through GitHub Issues:

<https://github.com/superheroYu/sshnet-share/issues>

Do not paste private keys, passwords, raw diagnostic bundles, access tokens, real server hostnames, usernames, or unredacted logs into public issues. Open a minimal issue with a non-sensitive summary and request private follow-up when details are sensitive.

## Diagnostics And Privacy

SSHNet Share does not automatically upload telemetry, crash reports, logs, or diagnostic bundles. Diagnostic ZIP files are generated locally and must be submitted manually by the user.
